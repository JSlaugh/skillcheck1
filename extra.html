<VirtualHost *:80>

DocumentRoot "/var/www/html/skillcheck1"

ServerName 52.87.224.215

Alias /aunt /var/www/html/skillcheck1/bobyouraunt

Alias /uncle /var/www/html/skillcheck1/bobyouruncle

Alias /nephew /var/www/html/skillcheck1/bobyournephew

</VirtualHost >

I added these lines of code to the /etc/httpd/conf/httpd.conf file

I wanted to make it so you could have alternate routes to the files instead of forcing the user to see the entire file structure

Not required in this context but useful later on potentially if I wanted to have two different applications running on the same server

So if you go to 52.87.224.215/uncle is routes to the uncle page

If you go to 52.87.224.215/aunt is routes to the aunt page

If you go to 52.87.224.215/nephew is routes to the nephew page
